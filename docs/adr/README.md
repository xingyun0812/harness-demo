# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for harness-demo.

## How to create a new ADR

Copy the template below:

```markdown
# ADR-{NNN}: {Title}

- **Status**: proposed | accepted | deprecated
- **Date**: {YYYY-MM-DD}
- **Deciders**: {names}

## Context

{Problem description and background}

## Decision

{What was chosen and why}

## Consequences

- {Positive: ...}
- {Negative: ...}
- {Neutral: ...}
```
